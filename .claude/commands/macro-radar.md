## Input dell'utente

Se non specificato, fornisci il quadro macro completo attuale.

## Istruzioni

### 1. Ricerca dati
Usa web search per trovare: dati macro più recenti (Fed, BCE, BOJ), indicatori economici, sentiment di mercato, eventi geopolitici.

### 2. Analisi Completa

#### A) Regime di Mercato
Determina il regime attuale:
- **Risk-On**: crescita forte, volatilità bassa, credito facile
- **Risk-Off**: recessione/paura, vola alta, flight to safety
- **Transizione**: segnali misti, cambio regime in corso
- **Stagflazione**: crescita bassa + inflazione alta
