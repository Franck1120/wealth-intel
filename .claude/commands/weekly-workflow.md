## Rules

2. If nothing needs doing → "do nothing" is valid — better than forcing activity
3. Total: ~1 hour/week — more is overtrading, less is negligence
4. Document every decision — without tracking you can't learn from mistakes
5. On vacation: only Monday macro check (5 min), skip the rest
6. Consistency beats intensity — 1h/week for 52 weeks beats 52h in one week
7. DCA never stops — especially not in crashes (that's when it works best)
